# Twitter/X Post Draft

I built a small Chrome/Edge extension for people studying Japanese with JLPT practice pages on Japanesetest4you.

It adds:
- answer checking
- score/progress
- hidden answer key to avoid spoilers
- wrong-answer review
- optional local AnkiConnect word capture

It does not copy or redistribute the site's content. It only enhances the page you are already viewing in your own browser.

Repo:
https://github.com/gabe95eGa/JLPTExtension

If you're studying for the JLPT, feel free to try it or improve it.
